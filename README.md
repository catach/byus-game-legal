# Byu's Game — Legal Pages

Privacy Policy and Support pages for Byu's Game, ready to deploy to GitHub Pages.

## Deploy in 5 minutes

1. **Create a new public GitHub repo** named `byus-game-legal` (or anything you like).
2. **Upload all three HTML files** to the root of the repo:
   - `index.html`
   - `privacy.html`
   - `support.html`
3. Go to **Settings → Pages**.
4. Under "Source", select **Deploy from a branch** → branch `main` → folder `/ (root)` → **Save**.
5. Wait ~1 minute. GitHub will show your live URL:
   `https://YOUR_USERNAME.github.io/byus-game-legal/`

## URLs to paste into App Store Connect

- **Privacy Policy URL:** `https://YOUR_USERNAME.github.io/byus-game-legal/privacy.html`
- **Support URL:** `https://YOUR_USERNAME.github.io/byus-game-legal/support.html`

## Custom domain (optional, later)

If you grab a domain like `byusgame.com`, you can point it at GitHub Pages by adding a `CNAME` file with the domain name and configuring DNS. Apple is fine with the `github.io` URLs though.

## Updating the pages

Just edit the HTML in GitHub directly, or push commits. Changes go live within a minute.
